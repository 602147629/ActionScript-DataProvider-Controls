Interfacepublic interface IDataRendererSkin extends ISkin, IDisplayObject, flash.events.IEventDispatcher

Data accepting programmatic skin interface.

Certain views accept skins, whose state depends on the current view data. Such a skin can only be recognised, if it implements this interface.

The skin owner will reset the skin data at any time its own data changes or a data property changes. The skin is then in charge to update its presentation.
