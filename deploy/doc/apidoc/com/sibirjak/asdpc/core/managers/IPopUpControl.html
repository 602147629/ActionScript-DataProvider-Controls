PopUpControl interface.

The IPopUpControl interface became necessary in conjunction with the style management. Styles set to a popup trigger button (e.g. the ColorPicker button or a SelectBox) should be available also to the particular popup (ColorPicker color list, SelectBox list). Since such popups are not direct children of their each trigger button, the interfaces IPopUpControl and IPopUpControlPopUp tell the StyleManager that style inheritance should respect also the trigger-popup relation.
